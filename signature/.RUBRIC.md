# Self-Review Rubric

Applied per-branch after draft + before declaring done. Use playwright: `file://` the built page, screenshot at 1440×900 and 390×844 (mobile), then score.

## Signal (positive)
- [ ] First impression: does it provoke a reaction within 3 seconds? (confusion, delight, curiosity all count)
- [ ] Has at least ONE element that would not appear on a typical portfolio site
- [ ] Interaction has personality — not just hover states, something that feels authored
- [ ] Typography / color / motion cohere into a deliberate voice
- [ ] Works on mobile (even if differently-weird on mobile)
- [ ] Loads without console errors

## Anti-signal (negative — iterate if any apply)
- [ ] Would this embarrass a design student? (generic cards, stock gradients, centered-column-with-sections layout)
- [ ] Could this be ChatGPT default output?
- [ ] Weirdness is cosmetic only (custom font on a normal layout) — weirdness must be structural or interactional
- [ ] Copy reads like a LinkedIn bio
- [ ] Uses emoji as personality substitute

## Content baseline (all branches)
Grace's identity: **Grace Wei-li Juan** — software engineer, polymath, visual artist. Content is placeholder-but-plausible: a few project cards, a blog teaser, an art section, contact link. Exact copy can be invented/absurd as long as the structural intent is legible.

## Decision rule
- All positive boxes + no anti-signal → **done, push, move on**
- Missing positive OR any anti-signal → iterate ONCE
- After iteration, regardless of score → **push and move on.** Iteration cap is hard.
